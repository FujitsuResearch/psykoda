Python versions¶

Python のインストールは tox が自動で行うのではなく、あらかじめ行っておいて tox に知らせる必要がある。

Windows¶

  • 環境変数 TOX_DISCOVER を tox.interpreters.common:base_discover が解釈する

  • tox のコマンドラインオプションで --discover C:\..\python.exe と フルパス を指定 (複数可) することができる

  • 環境を C:\Python38 などに作成すると tox.interpreters.windows:tox_get_python_executable が自動で見つける
